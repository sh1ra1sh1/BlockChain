# BlockChain
C++でブロックチェーンを実装した記事がなかったので勉強ついでに実装． 
© 2022 白石和暉

## 推奨環境
- GNU Make 4.3
- g++ (GCC) 11.2.0
## ビルド
`BlockChain`ディレクトリで以下の操作を行う．
```
make -j
```
## 実行
`BlockChain`ディレクトリで以下の操作を行う．
```
./blockchain
```
## 今後の展望
